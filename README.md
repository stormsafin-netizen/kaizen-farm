# 改善農園

`kaizen-farm.html` をブラウザで開くと起動します。

## できること

- 困りごとを「たね」として保存
- 6ステップで問題、影響、原因の仮説、改善案を整理
- 途中保存と再編集
- 改善提案サマリーとして収穫
- 収穫内容のコピー、保存、ブラウザ印刷
- JSON形式での書き出し、読み込み
- キャラクター表示と文字サイズの切り替え

## データについて

データは同じブラウザ内の `localStorage` に保存されます。外部AI API、ログイン、クラウド同期は使いません。

MVPではPDF専用生成は行わず、ブラウザ印刷を使います。キャラクターは正式素材ではなく、軽量な仮表示です。
